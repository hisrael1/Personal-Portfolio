Landbnb

Landbnb is a full-stack vacation rental application, inspired by Airbnb. The goal of this project was to replicate the core features of Airbnb while demonstrating proficiency in a range of web technologies. Built with React , Redux, Rails, PostgreSQL, JavaScript, HTML5, CSS3, AWS S3, and Google Maps API, the application is hosted on Heroku. Click on the image to be directed to the app.

Features

Users can search for vacation listings by location, available dates, and maximum number of guests. They can also host listings, upload photos stored in AWS S3, and input location data that gets geocoded by the Google Maps API for map display. The backend, powered by Ruby on Rails, serves as an API to implement CRUD functionality for Users, Listings, Bookings, and Reviews. I implemented User Authentication in Ruby by leveraging the Bcrypt gem for secure password hashing, and also provided a 'Demo User' option to enhance the user experience, allowing them to bypass the sign-up process and explore the app effortlessly.

CoinLook

CoinLook is a crypto price data visualization app that allows users to compare crypto price trends. This was my first project and the goal of this project was to become more familiar with the fundamentals of JavaScript and the Document Object Model before moving on to libraries such as React. Another goal of this project was to challenge myself to learn and implement D.3, a technology that I was unfamiliar with before starting this project. Built with JavaScript, HTML5, CSS3, and the D.3 library, the project is hosted on Github. Click on the image to be directed to the app.

Features

Users can select two coins as well as a start date and an end date. Once the user completes these fields and hits submit, the user will see a data visualization of the price trends being plotted concurrently. This is done by fetching data from the CoinGecko API using jQuery, parsing the data, and then appending the data to the DOM using a combination of event listeners, event handlers, and the D.3 library.

WeGo

WeGo is a full-stack social networking application designed to connect users interested in participating in similar activities. The goal of this project was to test my knowledge of software engineering fundamentals and ability to quickly learn and implement a new stack. Another objective of this project was to enhance my teamwork abilities in app development and refine my verbal communication skills, while mastering Git version control and related tools. This application was a group project built with MongoDB, Express, React, Redux, Node, AWS, HTML5, and CSS3. Click on the image to be directed to the app.

Features

Express middleware was implemented for efficient data transfer between the backend and frontend, enabling multi-criteria search functionality for users. The app features a robust data model with constraints designed and enforced in MongoDB, ensuring data consistency and integrity throughout the application. Integration of Redux with React allows for maintaining, validating, and persisting global state, while mapping state and dispatch into components via props.
